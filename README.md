A **machine learning-based Streamlit app** to detect whether a message or document is spam. It supports both **typed text messages** and **PDF documents**, providing a simple and interactive interface to classify spam.

---

## Features

- **Text Message Classification**: Type any SMS or email in the main area and check if it's spam.  
- **PDF Spam Detection**: Upload PDF files from the sidebar to check for spam content.  
- **Modern UI**: Stylish result cards, sidebar for PDF upload, and clear transformed text display.  
- **Text Preprocessing**: Automatic tokenization, stopword removal, and stemming for improved model accuracy.  
- **Fast Predictions**: Powered by a trained **Multinomial Naive Bayes** model.  

---

## Key highlights:

- **High Accuracy: Achieves 95.93% accuracy on test data.

- **High Precision: Maintains 100% precision, ensuring reliable spam detection.
